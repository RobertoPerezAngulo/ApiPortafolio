## download sqlLite DB

![Alt text](image.png)

```bash
# install dependencies
$ pipenv shell 

# install all pipenv environment 
$ pipenv install 

# install swagger 
$ pipenv install drf-yasg

# server with http://127.0.0.1:8000/ default 
$ pipenv run dev 
# or 
$ python manage.py runserver
```
